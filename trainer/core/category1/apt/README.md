* I'll show you how to use apt module of ansible
* Short documentation about command
```
Manages apt packages (such as for Debian/Ubuntu).

```
* Example
```
/opt/ansible/ansible# ansible localhost -m apt -a update_cache=yes
localhost | SUCCESS => {
    "cache_update_time": 1510591591, 
    "cache_updated": true, 
    "changed": true
}
```
